# 🎮 Adaptive AI-driven Curriculum Personalization Framework based on Real-time User Profiling

## 📌 Overview

The **Gamified AI-Based Curriculum Recommender** is an intelligent e-learning support system designed to tackle one of the biggest challenges in online education—**student disengagement and high dropout rates**.

By combining **Large Language Models (LLMs)** with **gamification principles**, the system dynamically generates **personalized, structured learning paths** tailored to each learner’s goals, skill level, and interests.

Unlike traditional recommender systems that rely on static content mapping or collaborative filtering, this project produces a **context-aware, multi-phase curriculum** that adapts to individual learners while keeping them motivated through game mechanics.

---

## 🚀 Key Features

* 🧠 **AI-Powered Curriculum Generation**

  * Uses LLM-based zero-shot reasoning to create customized learning roadmaps
* 📋 **Dynamic User Profiling**

  * Collects learner goals, proficiency, and interests through an adaptive questionnaire
* 🧩 **Hierarchical Learning Structure**

  * Multi-phase curriculum with progressive difficulty levels
* 🎮 **Gamification Layer**

  * XP (Experience Points) system
  * Achievement badges
  * Competitive leaderboards
* 📈 **Improved Engagement & Retention**

  * Designed to reduce cognitive overload and course attrition

---

## 🏗️ System Architecture

1. **User Interaction Layer**

   * Questionnaire-based data collection
2. **User Profile Vectorization**

   * Converts inputs into a high-dimensional learner profile
3. **Prompt Engineering Layer**

   * Structures prompts for deterministic, meaningful LLM output
4. **AI Curriculum Generator**

   * Produces a personalized, phase-wise curriculum
5. **Gamified Learning Environment**

   * Tracks progress, XP, badges, and rankings

---

## 🧪 Experimental Results

A pilot study with **10 participants** demonstrated:

* ✅ **91.5% curriculum relevance rating**
* 📊 **62% higher retention rate** compared to non-gamified learning systems
* 🔄 Reduced learner dropout and improved engagement
